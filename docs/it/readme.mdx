# Documentazione Agent Zero

Per iniziare con Agent Zero, segui i link qui sotto per guide dettagliate su vari argomenti:

- **[Installazione](/it/docs/installation):** Configura (o [aggiorna](/it/docs/installation#come-aggiornare-agent-zero)) Agent Zero sul tuo sistema.
- **[Guida all'Uso](/it/docs/usage):** Esplora le funzionalità GUI e gli scenari di utilizzo.
- **[Panoramica dell'Architettura](/it/docs/architecture):** Comprendi il funzionamento interno del framework.
- **[Contribuire](/it/docs/contribution):** Impara come contribuire al progetto Agent Zero.
- **[Risoluzione Problemi e FAQ](/it/docs/troubleshooting):** Trova risposte a problemi e domande comuni.

### La tua esperienza con Agent Zero inizia ora!

- **Scarica Agent Zero:** Segui la [guida all'installazione](/it/docs/installation) per scaricare ed eseguire Agent Zero.
- **Unisciti alla Community:** Unisciti alla community Agent Zero su [Skool](https://www.skool.com/agent-zero) o [Discord](https://discord.gg/Z2tun2N3) per discutere idee, fare domande e collaborare con altri contributori.
- **Condividi il tuo Lavoro:** Condividi le tue creazioni, workflow e scoperte di Agent Zero nella nostra area [Show and Tell](https://github.com/frdel/agent-zero/discussions/categories/show-and-tell) su GitHub.
- **Segnala Problemi:** Usa il [GitHub issue tracker](https://github.com/frdel/agent-zero/issues) per segnalare bug relativi al framework o suggerire nuove funzionalità.
