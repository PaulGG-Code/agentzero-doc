# Documentación de Agent Zero

Para comenzar con Agent Zero, sigue los enlaces a continuación para guías detalladas sobre varios temas:

- **[Instalación](/es/docs/installation):** Configura (o [actualiza](/es/docs/installation#cómo-actualizar-agent-zero)) Agent Zero en tu sistema.
- **[Guía de Uso](/es/docs/usage):** Explora las características de la interfaz gráfica y escenarios de uso.
- **[Panorámica de la Arquitectura](/es/docs/architecture):** Comprende el funcionamiento interno del framework.
- **[Contribuir](/es/docs/contribution):** Aprende cómo contribuir al proyecto Agent Zero.
- **[Solución de Problemas y FAQ](/es/docs/troubleshooting):** Encuentra respuestas a problemas y preguntas comunes.

### ¡Tu experiencia con Agent Zero comienza ahora!

- **Descarga Agent Zero:** Sigue la [guía de instalación](/es/docs/installation) para descargar y ejecutar Agent Zero.
- **Únete a la Comunidad:** Únete a la comunidad de Agent Zero en [Skool](https://www.skool.com/agent-zero) o [Discord](https://discord.gg/Z2tun2N3) para discutir ideas, hacer preguntas y colaborar con otros contribuidores.
- **Comparte tu Trabajo:** Comparte tus creaciones, flujos de trabajo y descubrimientos de Agent Zero en nuestra área [Show and Tell](https://github.com/frdel/agent-zero/discussions/categories/show-and-tell) en GitHub.
- **Reporta Problemas:** Usa el [GitHub issue tracker](https://github.com/frdel/agent-zero/issues) para reportar bugs relacionados con el framework o sugerir nuevas funcionalidades.
